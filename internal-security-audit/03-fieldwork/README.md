# 03 â€“ Fieldwork

This phase documents how audit testing was performed for selected controls. The fieldwork involves reviewing evidence, performing control walkthroughs, and evaluating the effectiveness of technical and procedural safeguards.

## Fieldwork Objectives

- Test control implementation and effectiveness
- Verify alignment with NIST CSF categories
- Document evidence reviewed and results
- Surface any compliance gaps, risks, or control failures

## Audit Techniques

- Control walkthroughs with stakeholders
- Technical configuration reviews
- Log and alert analysis
- Sampling of user access, incidents, and patch data

## Key Deliverables

- Completed control test scripts or checklist
- Fieldwork evidence collection (e.g., screenshots, logs, SOP excerpts)
- Audit notes and scoring (e.g., Pass/Fail, Risk Rating)
- Preliminary findings for each control tested
